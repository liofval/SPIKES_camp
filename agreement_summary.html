第1部　利用規約

第1条（目的・本規約の適用）

本規約は、SPIKES STAY PROJECT（以下「本プロジェクト」）が提供するホスト・ゲストマッチングサービス（以下「本サービス」）の利用条件を定めるものです。本サービスに参加するすべてのホスト・ゲスト（以下総称して「参加者」）は、本規約に同意したうえで利用するものとします。

第2条（定義）

用語定義
ホスト自己の居住する住居・シェアハウス等にゲストの宿泊を提供する参加者
ゲストホストの住居に宿泊する参加者
マッチング運営またはホスト・ゲスト間の合意によって宿泊組み合わせが成立すること
登録情報Googleフォーム・4S等を通じて参加者が提出した個人情報およびプロフィール情報
本イベントIVS Kyoto / SPIKESおよびそのサイドイベント

第3条（参加資格）

  1. 本サービスへの参加は、本イベントに参加する学生（大学生・大学院生・専門学校生）に限ります。
  2. 参加者は18歳以上であることを要します。18歳未満の場合は保護者の同意（別紙：保護者同意書）が必要です。
  3. 運営は、参加資格を満たさないと判断した場合、参加を拒否または取り消す権限を有します。

第4条（ホストの義務・禁止事項）

義務: 登録情報の正確な入力／マッチング成立後の事前連絡／安全な環境提供／断る場合の早期連絡。

禁止: 金銭・物品・サービスの要求／性的・暴力的・差別的言動／虚偽登録／第三者への無断個人情報提供／勧誘行為。

第5条（ゲストの義務・禁止事項）

義務: ホストルールの尊重／到着時間等の事前連絡／清潔保持・速やか退去／トラブル即時報告。

禁止: ホスト不安時の宿泊強行／第三者連れ込み／無許可飲酒・喫煙・違法薬物／個人情報のSNS無断公開／勧誘行為。

第6条（異性間マッチングの禁止）

安全性の観点から、異性間の宿泊マッチングは原則として禁止します。

第7条（宿泊中の行動規範）

  1. 夜22時以降は近隣配慮（窓を閉める／騒音回避）。
  2. 近隣等からの問い合わせには「友人が遊びに来ている」と対応し、本プロジェクト名を不必要に開示しない。
  3. 備品破損は速やかにホストへ告知・弁償。
  4. ゴミ処理はホストの指示に従う。

第8条（マッチングの性質・運営の関与）

マッチングは参加者間の自発的な合意に基づきます。運営が宿泊を保証するものではなく、不成立時は代替手段を可能な範囲で提案します。ホスト・ゲストとも理由を問わず断る権利・宿泊を中止する権利を有します。

第9条（免責事項）

運営は、参加者間で発生した紛争・トラブル・損害・事故等に対して、一切の法的責任を負いません。盗難・紛失・怪我・財物損害は当事者間で解決します。

第10条以降

個人情報の取扱いは第2部、参加資格の取消・規約変更・準拠法（日本法／京都地方裁判所）等は本書末尾に準じます。

第2部　プライバシーポリシー

1. 基本方針

運営は、参加者の個人情報を個人情報保護法その他関連法規を遵守し、適切に保護します。

2. 収集する個人情報

ホスト: 氏名/ニックネーム、大学・学年、最寄駅・町名（番地・号は収集しない）、最大受け入れ人数・受け入れ条件、属性／尖り、Discord ID／X ID。

ゲスト: 氏名/ニックネーム、大学・学年、居住地域、属性／尖り／Give、恩返し意向、訪問希望時期・泊数・目的、Discord ID／X ID、アレルギー等配慮事項（任意）。

3. 利用目的

マッチング、参加者間の情報共有、運営連絡、サービス改善（集計のみ）、安全性の確保。

4. 情報の閲覧範囲（マッチング目的）

マッチングを成立させる目的で、登録情報は以下の範囲で共有されます。
  1. マッチング相手候補（ホスト⇔ゲスト）への公開：プロフィール情報を相手側参加者に公開。
  2. SPIKES実行委員会・4S運営関係者による閲覧：登録データ全体を運営側が閲覧（守秘義務あり）。
  3. 連絡先（Discord ID / X ID）の開示：マッチング成立後に相手側へ開示。

5. 第三者提供

本人同意・法令・生命財産保護のため必要な場合を除き、第三者提供は行いません。

6. 管理・保護

Googleフォーム・スプレッドシート上で運営限定のアクセス権限のもと管理。本イベント終了から3ヶ月後を目安に削除。

7. 参加者の権利

開示請求・訂正/削除請求・利用停止請求が可能（Discord/Xから申請）。

第3部　重要事項同意チェックリスト

フォーム登録時に、以下のすべての項目に同意いただきます。

全参加者共通

  1. 運営は宿泊の安全・トラブルに責任を負わないことを理解した。
  2. 異性間の宿泊マッチングは禁止であることを理解した。
  3. 宿泊費・金銭の授受は禁止であることを理解した。
  4. マッチング相手（ホスト⇔ゲスト）に登録情報が公開されることに同意した。
  5. SPIKES実行委員会・4S運営関係者が登録データを閲覧することに同意した。
  6. マッチング成立後、Discord ID / X ID等の連絡先が相手に開示されることに同意した。
  7. 嫌だと感じたら断れる／断られる場合もあることに同意した。

ホスト専用

  1. 夜22時以降の近隣配慮を徹底し、本プロジェクト名を不必要に開示しない。
  2. 登録した最寄駅・町名はマッチング前から参加者に公開されることを理解した。

ゲスト専用

  1. ホストのルールを最優先に尊重する。
  2. マッチングが保証されるわけではなく、自己でゲストハウス等を手配する準備があることを確認した。
