SPIKES STAY PROJECT 利用規約

制定日：2025年●月●日
最終改定日：2025年●月●日
運営：SPIKES STAY PROJECT 実行委員会（以下「運営」）

第1条（目的・本規約の適用）

本規約は、SPIKES STAY PROJECT（以下「本プロジェクト」）が提供するホスト・ゲストマッチングサービス（以下「本サービス」）の利用条件を定めるものです。本サービスに参加するすべてのホスト・ゲスト（以下総称して「参加者」）は、本規約に同意したうえで利用するものとします。

本プロジェクトは、IVS Kyoto / SPIKES 参加学生を対象に、京都在住の学生（ホスト）と関西圏外から参加する学生（ゲスト）の宿泊マッチングを促進することを目的としています。

第2条（定義）

用語定義
ホスト自己の居住する住居・シェアハウス等にゲストの宿泊を提供する参加者
ゲストホストの住居に宿泊する参加者
マッチング運営またはホスト・ゲスト間の合意によって宿泊組み合わせが成立すること
登録情報Googleフォーム・4S等を通じて参加者が提出した個人情報およびプロフィール情報
本イベントIVS Kyoto / SPIKESおよびそのサイドイベント

第3条（参加資格）

  1. 本サービスへの参加は、本イベントに参加する学生（大学生・大学院生・専門学校生）に限ります。
  2. 参加者は18歳以上であることを要します。ただし、18歳未満の参加を認める場合には、保護者の同意を別途取得することを条件とします。
  3. 運営は、参加資格を満たさないと判断した場合、参加を拒否または取り消す権限を有します。

第4条（ホストの義務・禁止事項）

ホストが行うべきこと

  1. 登録情報（住所の町名・宿泊可能人数・プロフィール等）を正確に入力すること。
  2. マッチングが成立した場合、事前にゲストと直接連絡を取り、宿泊条件・到着時間・ルール等を合意すること。
  3. 宿泊中のゲストに対し、誠実かつ安全な環境を提供するよう努めること。
  4. 宿泊を断る場合は、マッチング確定前に速やかにゲストおよび運営へ連絡すること。

ホストの禁止事項

  1. 宿泊の対価として、いかなる名目であっても金銭・物品・サービスを要求すること（本サービスは無償が原則）。
  2. ゲストに対して性的・暴力的・差別的な言動を行うこと。
  3. 実際の住居状況と著しく異なる情報を登録すること（虚偽登録）。
  4. 運営の事前許可なく、宿泊者の個人情報を第三者へ提供すること。
  5. 宿泊を口実としたビジネス勧誘・マルチ商法・宗教勧誘等を行うこと。

第5条（ゲストの義務・禁止事項）

ゲストが行うべきこと

  1. ホストの住居のルール・生活習慣を尊重し、誠実に行動すること。
  2. 宿泊前にホストと直接連絡を取り、到着時間・宿泊期間・注意事項等を確認すること。
  3. 宿泊終了後は速やかに退去し、使用した場所を清潔に保つこと。
  4. 宿泊中にトラブルが発生した場合は、速やかに運営へ報告すること。

ゲストの禁止事項

  1. ホストが不安・恐怖を感じると申し出た場合に宿泊を強行すること。
  2. ホストの同意なく第三者を居住スペースへ連れ込むこと。
  3. ホスト宅における飲酒・喫煙（事前許可のある場合を除く）・違法薬物の使用。
  4. ホストの個人情報をSNS等で無断公開すること。
  5. 宿泊を口実としたビジネス勧誘・マルチ商法・宗教勧誘等を行うこと。

第6条（異性間マッチングの禁止）

安全性の観点から、異性間の宿泊マッチングは原則として禁止します。
  • ホストは同性のゲストのみを受け入れることを原則とします。
  • Googleフォーム登録時に受け入れ可能な性別を明示し、運営はこれを厳守してマッチングを行います。
  • 本規定に違反したマッチングを故意に行った参加者は、本サービスの利用資格を即時失います。

第7条（宿泊中の行動規範）

参加者は宿泊期間中、以下の行動規範を遵守するものとします。

  1. 夜22時以降は、近隣住民への配慮のため、窓を閉め、大声での議論・騒音を避けること。
  2. 近隣・管理会社・警察等から問い合わせを受けた場合は、「友人が遊びに来ている」という範囲で対応し、本プロジェクト名を不必要に開示しないこと。
  3. ホスト宅の設備・備品を丁寧に扱い、破損した場合は速やかにホストへ告知・弁償すること。
  4. ゴミの分別・処理はホストの指示に従うこと。

第8条（マッチングの性質・運営の関与）

  1. 本サービスにおけるマッチングは、運営が宿泊を保証するものではありません。マッチングはあくまで参加者間の自発的な合意に基づきます。
  2. マッチングが成立しなかった場合、運営は代替手段（バックアップホストの紹介・ゲストハウス情報の提供・マッチング失敗者同士でのAirbnb利用等）を可能な範囲で提案しますが、宿泊先の確保を保証するものではありません。
  3. ホストはウェイティングリストから自己の判断でゲストを選択する権利を有し、理由を開示せず申し込みを断ることができます。
  4. ゲストは、マッチング確定後であっても、ホスト宅到着後に危険・不安を感じた場合、宿泊を中止する権利を有します。この場合、運営は可能な範囲でサポートを行います。

第9条（免責事項）

  1. 運営は、ホスト・ゲスト間で発生した紛争・トラブル・損害・事故等に対して、一切の法的責任を負いません。
  2. 宿泊中に発生した盗難・紛失・怪我・財物損害等については、当事者間で解決するものとし、運営はいかなる補償も行いません。
  3. 参加者は、本サービスの利用に際して自己の責任と判断のもとで行動することを十分に理解したうえで参加するものとします。
  4. 運営は、登録情報の正確性・完全性を保証しません。虚偽の登録情報に基づいて発生したトラブルについて運営は責任を負いません。

第10条（個人情報の取り扱い）

参加者の個人情報の取り扱いについては、別途定める「プライバシーポリシー」に従います。参加者は、本規約への同意をもってプライバシーポリシーにも同意したものとみなします。

第11条（参加資格の取消・除外）

運営は、以下に該当する参加者の登録を取り消し、本サービスの利用を禁止できるものとします。

  1. 本規約に違反した場合。
  2. 他の参加者・第三者への迷惑行為・ハラスメントを行った場合。
  3. 虚偽の情報を登録した場合。
  4. その他、運営が不適切と判断した場合。

第12条（規約の変更）

運営は、必要に応じて本規約を変更できるものとします。変更後の規約は、運営が指定するWebページ等に掲載した時点から効力を生じます。重要な変更の場合は、参加者に別途通知します。

第13条（準拠法・合意管轄）

本規約は日本法に準拠します。本サービスに関連する紛争については、京都地方裁判所を第一審の専属的合意管轄裁判所とします。
