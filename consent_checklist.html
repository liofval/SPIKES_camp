重要事項同意チェックリスト

SPIKES STAY PROJECT — フォーム冒頭に掲載・全項目チェック必須。
このチェックリストは、ホスト・ゲストいずれのフォームにも冒頭で設置し、 全項目をチェックしないと次のページに進めないよう設定してください（Googleフォームの「必須」設定）。

全参加者共通以下の項目すべてに同意してください
01運営は宿泊の安全・トラブルに責任を負いません
本プロジェクトは参加者間の自発的な合意に基づくマッチングサービスです。 宿泊中に発生した盗難・怪我・トラブル・損害について、運営は一切の法的責任を負いません。 自己の判断と責任のもとで参加することに同意します。
02異性間の宿泊マッチングは禁止です
安全性の観点から、異性間の宿泊は禁止です。 ホストは同性のゲストのみを受け入れ、ゲストも同性のホストのみとマッチングされます。 この規則に違反した場合、即座に参加資格を失うことに同意します。
03宿泊費・金銭の授受は禁止です
ホストはいかなる名目であっても金銭・物品・サービスを要求することを禁止します。 本サービスは完全無償が原則であることを理解し、同意します。
04aマッチング相手（ホスト⇔ゲスト）に情報が公開されることへの同意
登録したプロフィール情報（ニックネーム・大学・学年・最寄駅／町名・属性・尖り・語りたいこと等）は、 マッチング相手候補となる参加者（ホストはゲスト全員、ゲストはホスト全員）に公開されます。 相手はこの情報を見てマッチングを判断します。
04bSPIKES実行委員会・4S運営関係者による閲覧への同意
マッチング調整・トラブル対応・安全管理のため、 SPIKES実行委員会および4S（Link）運営関係者が登録データ全体を閲覧します。 いずれも守秘義務を負います。
04cマッチング成立後の連絡先開示への同意
マッチングが成立した場合、登録したDiscord ID / X ID等の連絡先がマッチング相手に開示されます。 詳細はプライバシーポリシー第4条をご確認ください。
05嫌だと感じたらいつでも断れます・断られる場合もあります
ホストは理由を問わずゲストの宿泊を断ることができます。 ゲストも、現地到着後に不安を感じた場合は宿泊を中止できます。 マッチングは義務ではなく、お互いの合意が前提であることに同意します。
ホスト専用以下の項目にも同意してください
06宿泊中は近隣への配慮を徹底します
夜22時以降は窓を閉め、大声での会話・騒音を避けます。 近隣・管理会社から問い合わせがあった場合は「友人が遊びに来ている」と対応します。 本プロジェクト名を不必要に開示しないことに同意します。
07登録した住居情報（町名）は参加者に公開されます
登録した最寄駅・町名は、マッチング前の段階からゲスト参加者に公開されます。 番地・号・建物名は収集・公開されません。
ゲスト専用以下の項目にも同意してください
08ホストのルールを最優先に尊重します
ホスト宅では、ホストの生活ルールを最優先に従います。 喫煙・飲酒・第三者の無断連れ込み等はホストの明示的な許可なく行いません。 ホスト宅の設備を破損した場合は誠実に弁償します。
09マッチングが保証されるわけではありません
本サービスへの登録は、宿泊先の確保を保証するものではありません。 マッチングが成立しない場合、自己でゲストハウス・ホテル等を手配する準備があることを確認します。

すべての項目に同意いただいた方のみ、フォームへの回答に進んでください。
ご不明な点はDiscordサーバーの運営チャンネルまでお問い合わせください。

// Googleフォームでチェックボックスを必須にする実装例

// 全員共通チェックリスト
const consentItem = form.addCheckboxItem();
consentItem.setTitle("【必須】以下の重要事項すべてに同意します")
  .setHelpText("全項目にチェックを入れないと先に進めません")
  .setChoices([
    consentItem.createChoice("① 運営はトラブルに責任を負わないことを理解した"),
    consentItem.createChoice("② 異性間マッチングが禁止であることを理解した"),
    consentItem.createChoice("③ 宿泊費・金銭授受が禁止であることを理解した"),
    consentItem.createChoice("④ 個人情報が参加者間・運営間で共有されることを理解した"),
    consentItem.createChoice("⑤ マッチングはお互いの合意が前提で、断ることができることを理解した"),
  ])
  .setRequired(true);
